## 网站字体颜色的问题，终于搞定了

其实早就想改，一直拖着。今天终于推进了——每个 tab 的标题字体颜色一直不对，结果查来查去，问题出在 GitHub Actions 的任务队列里：有一个旧的 deploy 请求一直挂在那里没有完成，后面的 push 都被堵住了。

手动把它终断，重新 push，字体就乖乖变过来了。

感受：很多时候代码本身没问题，是流程里有个"挂起的旧任务"在阻塞，这种东西不往队列里看，怎么都找不到。

## 和山哥把 API 聊透了

今天和山哥在微信上深聊了一次 API，越聊越清晰。

我现在对 API 的理解是这样的：

**API 是服务提供方定义的接口规则。** 你想用它的服务，就得按它规定的格式提交请求，带上身份验证（Token），填好所需参数。格式不对，直接拒绝。

几个帮我想清楚的类比：

- **去政府窗口办事**：那张"申请表"就是 API。你得先填身份信息（Token），再填你要办的事项（参数），填完交给工作人员，他才会启动他的流程。表格填错，或者缺字段，就退回来让你重填。
- **手机的充电孔和耳机孔**：它们是手机向外部开放的 protocol interface，定义了电压、电流、信号格式。你用的耳机得符合这个接口，才能听到声音。
- **Mac 不支持触屏，iPad 支持**：这就是接口不同，决定了你只能用鼠标键盘跟 Mac 打交道。

山哥说："对于不写程序的人来说，目前的理解已经非常到位了。"

我把这句话存起来，算作一个小小的里程碑。

## 晚上看小 Lin：全球能源结构

几个让我印象深的数据：

- 中国能源结构里，原油约占 **13%**
- 霍尔木兹海峡输出的原油，超过 **60%** 运往中国
- 但中国能源结构多元：煤炭、LNG、清洁能源各占一块，原油并非命脉

所以伊朗-美国-以色列冲突，对中国的**直接**能源冲击有限，可控。

但间接影响更值得关注——韩国、日本高度依赖中东石油，它们受冲击，会传导到中国今年的外贸压力上。这条链路，比油价直接涨价的影响要深。
