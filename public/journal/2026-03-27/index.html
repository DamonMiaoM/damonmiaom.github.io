在这里疯狂地记日记，Document Relentlessly ！！！

不同于 Blog 里的内容，Journal 中的都是日常生活。Claude Code 给了我想象的空间，未来我想把 Vlog 或者其他日常生活的记录放到这里。

反观 Blog，我现在都是让 Claude Code 帮我完成一项任务之后，wrap up 成一小篇技术性的文章，在我看来它本质上是 Learning Log。在 Claude Code 的加持下，完整地把搭建网站的过程记录下来变得非常轻松。虽然很多技术性的内容目前我还看不懂，不过因为有了体验，拿到结果之后，再反过来重新检查流程。每一步采取的 action 到底意味着什么，交给 Claude 来解释。

对，先拿到结果，再学习。

昨天吃饭、聊天到很晚，不过并不妨碍我带着强大的热情一早起来就跟 Claude Code 继续捣鼓个人网页。

今天添加了 Changelog，非常有意思，一旦进入开发者的视角和思维，一个我未曾想象的想象空间瞬间打开。

AI 时代的创造路径 ： 用心记录生活、体验、感受、想法，沉淀下来，形成个人数据资产（Personal Data Assets），喂给 AI，让他们帮忙形成洞见 Insights，既加速个人成长，还能够形成个体化的产品，可谓一举两得。
